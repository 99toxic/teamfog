Adventure Team Views

Mount Pisgah W4C/CM-011

Mount Pisgah Summit altitude is 5721’ This is a 1.5 mile hike with 700’ elevation gain. There is a nice observation deck at the summit providing spectacular views. This is a very popular site and can get extremely crowded on the weekends.

Max Patch Mountain W4C/CM-036

Max Patch Mountain Summit altitude is 4629’ This is a ½ mile hike with 300’ elevation gain. This is an easy hike with 360 degree ~ unbelievable views. *Note this is a very popular site, and it can get extremely crowded in the afternoon.

High Rock W4T/SU-024

High Rock Summit altitude is 4500’ This is a 1.5 mile hike with 350’ elevation gain. This is a very nice hike with some rolling hills (ups & downs) to the summit. Great hike to get away from it all. This one of SOTA Adventure Team FOG’s favorite hikes.

Table Rock Mountain W4C/EM-051

Table Rock Mountain Summit altitude is 3940’ This is a .7 mile hike with 510’ elevation gain. This is a fairly steep hike, but the views are outstanding. This looking over the Lineville Gorge Wilderness Area.
